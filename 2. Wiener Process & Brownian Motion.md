`# Wiener Process & Brownian Motion
## Definition A **Wiener process** (or **Brownian motion**)
$W_t$ over $[0, T]$ satisfies: 
1. $W(0) = 0$. 
2. Increment $W(t) - W(s) \sim N(0, t-s)$ for $0 \leq s < t \leq T$. 
3. Non-overlapping increments are independent. **Properties**: - $E[W(t)] = 0$, $E[W(t)^2] = t$. - Covariance: $E[W(t)W(s)] = \min(t, s)$. 

--- 
## Simulation of Wiener Process 
### Discrete Approximation 
- Divide \([0, T]\) into intervals of length $\Delta t$. - $W_{i+1} = W_i + \sqrt{\Delta t} \, N(0,1)$. **Pseudocode**: 
```python 
Δt = T / L W[0] = 0 for i in 0 to L-1:     
ξ_i = sample from N(0,1)    
W[i+1] = W[i] + √(Δt) * ξ_i
````

## Refining the Path

To add intermediate points (e.g., ( $W(t_{i+1/2})$ )):  $W(t_{i+1/2}) = \frac{1}{2}(W(t_i) + W(t_{i+1})) + \frac{1}{2}\sqrt{\frac{\Delta t}{2}} , \epsilon_{i+1/2}, \quad \epsilon_{i+1/2} \sim N(0,1)$ 

**Generalization**: For ( $\alpha \in (0,1)$), define: $W((1-\alpha)t_i + \alpha t_{i+1}) = (1-\alpha)W(t_i) + \alpha W(t_{i+1}) + \sqrt{\alpha(1-\alpha)\Delta t} , \epsilon$

---

## Scaling Property

If ( $W(t)$ ) is Brownian motion, then for ( $c > 0$): $V(t) = \frac{1}{\sqrt{c}} W(ct) \quad \text{is also Brownian motion.}$

---

## Non-Differentiability & Unbounded Variation

- **Intuition**: ( W(t) ) is nowhere differentiable.
- **Variation**: Infinite for Brownian motion (unlike differentiable functions).

**Proof Sketch**:

- For ( $f \in C^1[0,1]$ ), variation is bounded.
- For ( $W(t) ), ( \sum (\Delta W_i)^2 \to T$ ) as ( $\Delta t \to 0$), but ($\max |\Delta W_i| \to 0$ ).

---

## Connection to White Noise

- **Formal derivative**: ( $\frac{dW}{dt} = N(t)$ ) (white noise).
- **Heuristic**: ( $E[N(t)] = 0 ), ( E[N(t)N(s)] = \delta(t-s)$ ).
